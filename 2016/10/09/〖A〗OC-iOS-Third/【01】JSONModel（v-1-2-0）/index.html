前言

JSONModel 是一个类，它允许你快速的创建智能的数据模型（model），你可以在 iOS 和 OS X 应用中使用它。

了解 JSONModel

JSONModel 可以自动和你的 JSON 进行类型匹配（如果 JSON 的数据类型跟你想要的类型不一样，可以转化为你想要的类型），大大减少代码量。

JSON 的数据格式通常只有有中间的部分：string、number、 array、object、null。
